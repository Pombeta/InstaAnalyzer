📌 InstaAnalyzer

Projeto em Java que tinha como objetivo analisar seguidores e identificar quem não segue de volta no Instagram.

⚠️ Status do projeto

Atualmente o projeto não funciona mais completamente, pois o Instagram restringiu o acesso a esse tipo de informação através de sua API e também bloqueou métodos de consulta automatizada de seguidores.

🎯 Objetivo
Praticar Java
Manipulação de listas e dados
Simular análise de seguidores
Entender integração com APIs externas
