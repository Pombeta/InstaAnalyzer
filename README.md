# 📌 InstaAnalyzer

Projeto em Java que tinha como objetivo analisar seguidores e identificar quem não segue de volta no Instagram.

---

## ⚠️ Status do projeto

Atualmente o projeto não funciona mais completamente, pois o Instagram restringiu o acesso a esse tipo de informação através de sua API e também bloqueou métodos de consulta automatizada de seguidores.

---

## 🎯 Objetivo

- Praticar Java  
- Manipulação de listas e dados  
- Simular análise de seguidores  
- Entender integração com APIs externas  

---

## 🚀 Como executar o projeto

Para rodar o InstaAnalyzer, é necessário fornecer um arquivo de dados com informações exportadas da sua conta do Instagram (seguidores e seguindo).

⚠️ Observação:  
O projeto não se conecta diretamente ao Instagram. Ele utiliza arquivos locais com os dados para realizar a análise.

### Passos:

1. Baixe seus dados do Instagram (seguidores e seguindo)
2. Coloque os arquivos na pasta do projeto
3. Execute a classe principal `Main.java`
4. O programa irá comparar as listas e mostrar quem não segue de volta
